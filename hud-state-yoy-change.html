Year to Year Changes in County-level Housing Costs

An application of R Shiny and the HUD API

Using data from Housing and Urban Development API, I created this to easily create graphics to compare Year to Year percent changes in housing costs by county for all States except for Hawaii and Alaska. The user can specify the unit type being compared the the beginning and end year. The tool will generate and image and table for every query. The user can choose to download the image generated to their size and resolution preference.
