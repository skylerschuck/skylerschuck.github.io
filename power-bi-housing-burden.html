Colorado Housing Burden

An application of Power BI

Using data from Census American Community Survey (ACS) 5-year estimates, I created this interactive dashboard to show the percentage of housing units of a given county that are burdened by housing costs >30% of their monthly income. The dashboard was made using the Census ACS API for the most accurate data.

I created this dashboard for the Regional Economic Development Institute (REDI) at Colorado State University. Please check out our other awesome work at csuredi.org
